Ad Channel Monitoring

Scope

Ad channel monitoring and audience targeting rules engine — integrating with YouTube and Google Ads APIs to keep brand advertising within defined audience boundaries.

Full Stack BÆRSkin Tactical · DivBrands Production Microservice
◈ Overview

Scope is a full-stack marketing monitoring application that integrates with the YouTube API and Google Ads API to track the channels where ads are being shown. Based on configurable audience targeting rules, it automatically identifies channels that fall outside the defined target audience and excludes them from future ad placements.

The project demonstrates practical knowledge of ad platform APIs, data collection pipelines, and building internal tools that give marketing teams operational control over campaign placement at scale.

◈ Features
  • API for receiving operations requests from marketing departments
  • Internal UI to review channel data and submit targeting rule changes
  • Background workers that continuously collect data from the Google Ads and YouTube APIs
  • Automated channel exclusion operations via the Google Ads API based on audience rules
◈ Architecture
System Context
Part of a broader microservices ecosystem. Scope operates as an independent monitoring service — ingesting data from ad platforms and applying targeting rules without depending on other internal services.
Frontend
Internal UI for the marketing team to review channel placement data, inspect flagged channels, and submit or configure audience targeting rules.
Backend API
Handles frontend requests and exposes endpoints for rule configuration, channel review, and triggering manual operations against the Google Ads API.
Async Workers
Custom background workers collect channel and campaign data from YouTube and Google Ads on a schedule, then evaluate placement against targeting rules.
Database
PostgreSQL stores channel records, targeting rules, placement history, and exclusion logs for reporting and auditing.
